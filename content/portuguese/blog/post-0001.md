Passo a passo para utilizar o sensor DS18B20 com o Raspberry

## Materiais necessários:

- Raspberry Pi
- Sensor de temperatura DS18B20
- Resistor de 4,7KΩ
- Placa de ensaio (opcional)

![Diagrama de conexão](/images/gallery/post-0001/1_o5k1z9aPwSp0rWeI0BjTYQ.webp)

## Compreendendo o sensor DS18B20

O DS18B20 é um sensor digital de temperatura conhecido por sua precisão e facilidade de uso. Sua interface de um fio simplifica o processo de conexão, tornando-o um favorito entre amadores e profissionais. Com sua capacidade de operar em uma ampla faixa de temperatura, é adequado para diversas aplicações, desde automação residencial até monitoramento industrial.

## Configurando Raspberry Pi

Antes de mergulhar na conexão do sensor, certifique-se de que seu Raspberry Pi esteja configurado e funcionando. Você pode instalar o sistema operacional Raspbian, conectar-se à internet e habilitar a interface SSH para acesso remoto. Além disso, certifique-se de que seu Raspberry Pi tenha os pinos GPIO necessários disponíveis para conectar o sensor DS18B20.

## Requisitos

No sistema Linux onde seu Node-RED está rodando e onde seus sensores estão conectados, certifique-se de ter carregado todos os módulos do kernel necessários para trabalhar com dispositivos 1-Wire, que é o sensor DS1820. Assim, você precisa ativar ou ter ativado a Interface 1-Wire no Rpi. Para fazer isso basta acessar a página Raspi-Configuração com:

```bash
sudo raspi-config
```

Vá para **Interfaces** e depois para **1-Wire**, habilite-o e reinicie o dispositivo. Você terminou.

## Plugins Node-RED necessários:

- node-red-dashboard
- node-red-contrib-ds18b20-sensor
