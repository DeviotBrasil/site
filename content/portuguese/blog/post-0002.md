Passo a passo para instalação do Nodered no Raspberry Pi via Docker

![Nodered + Docker + Raspberry](/images/gallery/post-0002/1_TWqP4JHjcrAl0rmMh8uIpw.webp)

Atualize o SO do seu Raspberry:

```bash
sudo apt update
```

Instale o Docker executando o comando:

```bash
curl -sSL https://get.docker.com | sh
```

ou

```bash
sudo apt install docker.io
```

Adicione seu usuário ao grupo Docker, para evitar ter que usar sudo toda vez que você executar um comando Docker:

```bash
sudo usermod -aG docker $USER
```

Use o comando docker run para iniciar o container do Node-RED com acesso aos GPIOs. O acesso aos GPIOs geralmente requer acesso ao dispositivo /dev/gpiomem.

Execute o seguinte comando:

```bash
sudo docker run -d --name mynodered --restart always --privileged -p 1880:1880 -v node_red_data:/data --device /dev/gpiomem nodered/node-red:latest
```

Ativando a Interface 1-Wire no Rpi. Para fazer isso basta acessar a página Raspi-Configuração com:

```bash
sudo raspi-config
```

Vá para **Interfaces** e depois para **1-Wire**, habilite-o e reinicie o dispositivo.
